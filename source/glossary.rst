.. glossary::

    WSGI
        Web Server Gateway Interface. It is a specification for web servers
        and application servers to communicate with web applications
        (though it can also be used for more than that). It is a Python
        standard, described in detail in PEP 333.

    Basic Authentication
       The client authenticates by sending an *Authorization* header of the
       form: "Basic base64(username + ':' + password)
